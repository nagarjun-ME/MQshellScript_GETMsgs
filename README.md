# MQshellScript_GETMsgs

This script will read messagsed from MQ queue 

The MQ parameters are 
=====================================

QMGR=TESTQM01

export OQName_1=TEST.INQ

export HOST=localhost

export CHL=SECURE.APP.CHANNEL

export PORT=1414

Note : You can change your parameters
